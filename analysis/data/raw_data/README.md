## Variables

The dataset contains the following variables for 32 automobile models:

| Variable | Units | Description |
|----------|-------|-------------|
| `mpg` | miles/gallon | Miles per US gallon (fuel efficiency) |
| `cyl` | cylinders | Number of cylinders (4, 6, or 8) |
| `disp` | cu.in. | Engine displacement (cubic inches) |
| `hp` | hp | Gross horsepower |
| `drat` | ratio | Rear axle ratio |
| `wt` | 1000 lbs | Vehicle weight |
| `qsec` | seconds | 1/4 mile acceleration time |
| `vs` | type | Engine type (0 = V-shaped, 1 = Straight) |
| `am` | type | Transmission (0 = Automatic, 1 = Manual) |
| `gear` | gears | Number of forward gears (3, 4, or 5) |
| `carb` | carburetors | Number of carburetors |

## Data Limitations

### Temporal
- **Publication Date**: 1974
- **Applicability**: Modern vehicles behave differently (hybrids, EVs, turbocharged engines)

### Sample
- **Size**: Only 32 vehicles (limited statistical power)
- **Scope**: Primarily American and imported passenger cars
- **Exclusions**: No commercial vehicles, trucks, or SUVs

### Statistical
- **Multicollinearity**: Weight and displacement are highly correlated
- **Small n**: Confidence intervals should be interpreted cautiously
- **Snapshot**: Single time point; no longitudinal variation
